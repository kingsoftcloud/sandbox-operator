# Sandbox Operator deploy resources

This directory contains the plain Kubernetes manifests required to run the
operator:

- Namespace
- CRDs
- RBAC
- operator ConfigMap
- webhook TLS Secret generated by `scripts/deploy.sh`
- manager Deployment
- webhook Service, MutatingWebhookConfiguration, and ValidatingWebhookConfiguration

Apply order:

```bash
kubectl apply -f 00-namespace.yaml
kubectl apply -f 01-crd.yaml
kubectl apply -f 02-rbac.yaml
kubectl apply -f 03-config.yaml
# create sandbox-operator-webhook-server-cert and patch webhook caBundle;
# scripts/deploy.sh does this with openssl.
kubectl apply -f 04-manager.yaml
kubectl apply -f 05-webhook.yaml
```

Notes:

- `scripts/deploy.sh` generates a self-signed CA and serving certificate for the webhook, stores the serving keypair in `sandbox-operator-webhook-server-cert`, then patches `caBundle` into the MutatingWebhookConfiguration and ValidatingWebhookConfiguration.
- cert-manager is not required for the plain manifest deployment.
- `03-config.yaml` contains KOP OpenAPI defaults: `Service=aicp`, `Version=2026-04-01`, `OPENAPI_AUTH_MODE=kop-sigv4`.
- Business namespaces still need their own OpenAPI AK/SK Secret. The default name is configured by `DEFAULT_OPENAPI_CREDENTIAL_SECRET`; see `config/credentials/credentials.example.yaml`.

Build and deploy:

```bash
docker build -t sandbox-operator:latest .
IMAGE=sandbox-operator:latest ./scripts/deploy.sh
```

Helm deploy:

```bash
helm install sandbox-operator charts/sandbox-operator \
  -n sandbox-operator-system \
  --create-namespace \
  --set image.repository=sandbox-operator \
  --set image.tag=latest
```

The chart does not render a `Namespace` object by default. Use Helm
`--create-namespace`, or create the namespace before installing. If your Helm
platform does not support `--create-namespace`, set `createNamespace=true` only
for a namespace that does not already exist.

The Helm chart defaults to an in-chart self-signed webhook certificate. To use
cert-manager instead, install cert-manager in the cluster and set:

```bash
helm install sandbox-operator charts/sandbox-operator \
  -n sandbox-operator-system \
  --create-namespace \
  --set certManager.enabled=true \
  --set webhook.selfSigned.enabled=false
```
